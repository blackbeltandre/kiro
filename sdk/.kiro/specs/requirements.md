# Witch Chat Interface - Requirements

## Overview
Create a mystical witch-themed chat interface using HTML and CSS only.

## User Stories
1. As a user, I want to see a list of participants on the left sidebar
2. As a user, I want to see a witch character with a glowing orb in the center
3. As a user, I want to see chat messages and offerings to the witch
4. As a user, I want to see a document/skull icon section on the right
5. As a user, I want to input my thoughts and chat with the witch

## Acceptance Criteria
- Dark mystical theme with forest/magical background
- Left sidebar with "Peserta videall" (participants) list
- Center area with witch illustration holding glowing orb
- Chat bubbles showing conversation with the witch
- Right panel showing "Offerings to the witch" with document upload
- Input area for "Page Thoughts" with text input
- Mobile-responsive layout
- Pure HTML and CSS only (no JavaScript)

## Design Elements
- Dark green/teal mystical background
- White/cream colored UI cards
- Witch illustration as central focal point
- Crystal ball/orb with glow effect
- Gothic/mystical typography
